# Microsoft 365 E3 Enterprise SharePoint Lab ☁️🏢

An enterprise-scale cloud tenant testing environment focused on SharePoint site architecture, front-end design, and granular identity access management.

## 📄 Project Overview
This repository documents the deployment of a Microsoft 365 E3 trial tenant used to explore the front-end capabilities of SharePoint Online. The project focuses on building collaborative environments, structuring document libraries, and enforcing security baselines to mimic a corporate intranet.

## 🔍 Key Lab Objectives & Configurations
* **Front-End Site Design:** Built functional communication and team sites to evaluate user experience (UX), navigation flows, and web part layouts.
* **Information Architecture:** Engineered structured document libraries with custom metadata columns to organize enterprise data efficiently.
* **Access Control & Permissions:** Tested permission inheritance, unique library permissions, and secure external sharing configurations to validate data protection boundaries.
